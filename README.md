# Government-Services RAG Chatbot (RTI Assistant)

This project is a Retrieval-Augmented Generation (RAG) chatbot built for SMAI Assignment 3 (Theme 10.3). It assists users by answering questions related to the Right to Information (RTI) Act in India, grounded solely on the provided official PDF documentation.

## Tech Stack

- **Frontend Framework:** Streamlit
- **Document Parsing:** PyPDF
- **Embeddings:** `sentence-transformers/all-MiniLM-L6-v2` (Running locally on CPU)
- **Vector Database:** ChromaDB
- **LLM:** Google Gemini (configurable via `.env`, default: `gemini-2.0-flash`) (via LangChain)
- **Orchestration:** LangChain

## Repository Structure

```
.
├── app.py               # Main Streamlit Chat Application
├── ingest.py            # Script to parse PDFs, chunk text, and persist embeddings
├── requirements.txt     # Python dependencies
├── .env                 # Environment variables (Add your GOOGLE_API_KEY here)
├── data/                # Directory containing RTI PDF documents
└── chroma_db/           # Local vector store generated by ingest.py
```

## Setup Instructions

### 1. Install Dependencies

Make sure you have Python installed. Run the following command to install required packages:

```bash
pip install -r requirements.txt
```

### 2. Configure Environment

1. Open the `.env` file in the root directory.
2. Replace `your_gemini_api_key_here` with your actual Google AI Studio API key.
3. (Optional) Set `GEMINI_MODEL` if you want to override the default model.

Example `.env`:

```env
GOOGLE_API_KEY=your_gemini_api_key_here
GEMINI_MODEL=gemini-2.0-flash
```

### 3. Ingest Data

Before running the application, you must parse the PDFs and build the vector database:

```bash
python ingest.py
```

This will read the PDFs in the `data/` folder, generate embeddings, and save them in the `chroma_db/` directory.

### 4. Run the Streamlit App

Start the Streamlit chatbot application:

```bash
streamlit run app.py
```

This will open the web app in your browser where you can ask questions like:

- "How do I file an RTI online?"
- "What is the fee for an RTI application?"

## Features

- **Accurate Responses:** Uses advanced RAG techniques to ground LLM answers in actual RTI document context.
- **Source Citations:** Appends source document names and page numbers to every generated answer.
- **Chat Interface:** Provides an intuitive conversational UI with session memory.
